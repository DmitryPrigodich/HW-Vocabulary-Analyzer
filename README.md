# HW-Vocabulary-Analyzer
Project is made to analyze the words and names of Kushan language


## Project creation
```
dotnet new classlib -n Tests
cd Tests
```


## Reqnroll
```
dotnet add package Reqnroll
dotnet add package Reqnroll.Tools.MsBuild.Generation
```

## Playwright
```
dotnet add package Microsoft.Playwright
```

## CSV
```
dotnet add package CsvHelper
```
